# GuffGaff

[Visit](https://ggaff.onrender.com/)

## Features

- User authentication
- Discussion creation and involvement
- Discussion, comments & replies
- Notifications

## Technology Stack

### Frontend
- Angular
- TypeScript
- Bootstrap

### Backend
- Web API
- SQL Server
- PostGreSQL

## Project Status

⚠️ This project is a personal learning and portfolio project. It is functional but still under development and not intended for production use.

## Getting Started

### Prerequisites

Before running the application, ensure you have the following installed:

* .NET SDK (version X.X or later)
* Visual Studio 2022 (or later)
* Node.js and npm
* Angular CLI

### Installation

#### 1. Clone the Repository

```bash
git clone https://github.com/dahalsweekar/GuffGaff
cd GuffGaff
```

#### 2. Set Up the Frontend (Angular)

Navigate to the UI project folder:

```bash
cd ui
```

Install the required dependencies:

```bash
npm install
```

Start the Angular development server:

```bash
ng serve
```

The frontend will be available at:

```text
http://localhost:4200
```

#### 3. Set Up the Backend (.NET)

Open the solution file (`.sln`) located in the backend folder using Visual Studio.

Restore the required NuGet packages:

```bash
dotnet restore
```

Run the backend application:

* Press **F5** in Visual Studio, or
* Execute:

```bash
dotnet run
```

### Running the Application

1. Start the backend application.
2. Start the Angular frontend.
3. Open your browser and navigate to:

```text
http://localhost:4200
```

### Project Structure

```text
project-root/
│
├── ui/                # Angular frontend
├── backend/           # .NET backend
├── db/              # Test Data
└── README.md
```

### Notes

* Ensure the backend service is running before using the frontend.
* Update the API base URL in the Angular configuration if required.
* This project is currently under active development and may contain unfinished features.
